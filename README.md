# Tech Fusion 2K25

## Description
Developed a departmental fest website for MCA 1st and 2nd-year students, featuring event details, registration, and an interactive UI.

## Features
- Fest information hub
- Online registration system
- Dynamic animations
- Responsive design

## Tech Stack
- HTML
- Tailwind CSS
- React.js
- TypeScript
- JavaScript
- Framer Motion
